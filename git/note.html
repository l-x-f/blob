                            # git 常用命令笔记

                            # 一.安装配置

                            1.配置个人的用户名称和电子邮件地址：

                            git config --global user.name "your name"
git config --global user.email "your email"
                            1
                            2

                            2.查看配置信息

                            git config --list
                            1

                            3.生成SSH Key （一路回车就行）

                            sh-keygen -t rsa -C "youremail@example.com"
                            1

                            # 二.创建仓库

                            git init
                            1
                            git clone <repo> <directory>
                            1

                            参数说明：

                            repo:Git 仓库。

                            directory:本地目录。

                            # 三.基本操作

                            1.文件添加到缓存区

                            缓存所有

                             git add .
                            1

                            缓存指定文件

                            git add  filename
                            1

                            2.文件修改后未执行git add 操作想恢复原样

                            恢复所有

                            git checkout   .
                            1

                            恢复指定文件

                            git checkout  filename
                            1

                            3.文件修改后执行了git add 操作想恢复原样 即 取消之前的 git add 添加

                            恢复所有

                            git reset .
                            1

                            恢复指定文件

                            git reset filename
                            1

                            4.查看修改状态

                            查看上次提交之后是否有修改。

                            git status   or   git status -s  # 简短的结果输出
                            1

                            查看执行 git status 的结果的详细信息。

                            git diff
                            1

                            尚未缓存的改动：git diff

                            查看已缓存的改动： git diff --cached

                            查看已缓存的与未缓存的所有改动：git diff HEAD

                            显示摘要而非整个 diff：git diff --stat

                            5.将缓存区内容添加到仓库中

                            git commit -m  (message)
                            1

                            跳过git add 提交缓存的流程

                            git commit -a  # 然后进入vim编辑器写信息
                            1

                            6.执行git commit 后修改提交信息

                            git commit --amend # 然后进入vim编辑器修改
                            1

                            7.版本回退

                            回退到上一个版本

                            git  reset --hard HEAD^
                            1

                            回退到任意版本（git log --oneline 或者 git reflog 查看版本号）

                            git  reset --hard HEAD~100  # HEAD~100 :版本号
                            1

                            8.代码提交

                            添加远程仓库

                            git remote add [shortname] [url]  # 常用 git remote add origin  [url]
                            1

                            查看远程仓库

                            git remote -v
                            1

                            远程仓库的重命名 （origin 改成 old_origin）

                            git remote rename origin old_origin
                            1

                            远端仓库origin的移除

                            git remote rm origin
                            1

                            提交代码并关联仓库和分支

                            git push -u origin master # 以后执行 git push 就可以了
                            1

                            # 四.分支操作

                            1.新建

                            git branch (branchname)
                            1

                            2.切换

                            git checkout (branchname)
                            1

                            3.查看所有分支

                            git branch -a
                            1

                            4.删除分支

                            git branch -d (branchname)
                            1

                            5.其他分支合并到当前分支

                            git merge (otherbranchname)
                            1

                            # 五.标签操作

                            1.给某个版本打标签（发布前）

                            git tag -a (tagname)  # 标签名
                            1

                            2.给某个版本打标签（发布后）

                            git tag -a  HEAD~  # 版本号 git log --oneline 或者 git reflog 查看版本号）
                            1

                            3.删除标签

                            git tag -d (tagname)   or   git tag -d HEAD~
                            1

                            4.查看标签

                            git tag
                            1
